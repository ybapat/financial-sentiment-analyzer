Behind the Scenes

Listening Trader is an AI-powered web app that analyzes market sentiment from Reddit. Here's how it works under the hood:

  • Data Collection: A Python backend uses PRAW to scrape posts and comments from finance-related subreddits every hour.
  • Ticker Detection: Comments are scanned for stock and crypto tickers, filtering out common words to avoid false positives.
  • Sentiment Analysis: Each comment mentioning a ticker is analyzed using a machine learning model trained on financial text.
  • Aggregation & Storage: Results are aggregated and stored in a database for fast, real-time querying by the frontend.
  • Frontend Visualization: The web UI displays sentiment, mentions, and historical trends with interactive charts and vibrant visuals.
